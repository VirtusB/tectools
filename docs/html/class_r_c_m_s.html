Detailed Description

Class RCMS Denne klasse er entry-pointet for enhver webside der bruger RCMS-systemet. Klassen står for at forbinde til MySQL og for at loade andre nødvendige klasser. Den indeholder "execute" metoden, som står for at eksekvere SQL på en sikker måde, ved at benytte parametre og Prepared Statements
