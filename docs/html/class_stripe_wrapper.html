Detailed Description

Class StripeWrapper Denne klasse fungerer som en wrapper til Stripe API'et. Den indeholder metoder som gør det nemmere at arbejde med API'et og reducere mængden af kode der skal skrives. Den indeholder bl.a. metoder til at oprette, redigere og slette kunder og hente produkter
